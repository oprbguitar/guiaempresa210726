# Nexo Industrial Perú

Primera versión pública de una plataforma visual para organizar normativa, herramientas de ingeniería industrial, análisis empresarial y mejora continua.

## Sitio publicado

https://oprbguitar.github.io/guiaempresa210726/

## Funcionalidades actuales

- Interfaz responsive inspirada en la última propuesta visual.
- Paleta de alto contraste: negro, marfil, naranja y magenta.
- Roadmap interactivo de seis fases.
- Tarjetas de SST, gestión laboral, procesos y datos.
- Ventanas informativas para cada módulo.
- Navegación móvil y comportamiento accesible por teclado.
- Ilustración industrial construida en SVG.
- Despliegue automático mediante GitHub Actions.

## Próximos módulos

- Normativa peruana.
- SST y SCTR.
- Legislación laboral.
- Kaizen, 5S y VSM.
- Producción y logística.
- Calidad y riesgos.
- Mantenimiento y activos.
- Analítica empresarial.
- Recursos académicos y tesis.

## Desarrollo local

El proyecto es estático y no requiere compilación.

```bash
python -m http.server 8080
```

Luego abra `http://localhost:8080`.

## Aviso

El contenido es informativo y referencial. La normativa aplicable debe verificarse en fuentes oficiales vigentes antes de tomar decisiones.
